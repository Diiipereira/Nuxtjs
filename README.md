# darkMode

## Build Setup

```bash
# instalar as dependências
$ npm install
$ npm install --save-dev @nuxtjs/color-mode
$ npm install --save-dev @nuxtjs/svg

# Iniciar o servidor em localhost:3000
$ npm run dev

# build para produção e iniciar servidor
$ npm run build
$ npm run start

# Gerar projeto estático
$ npm run generate
```
